# Unleashed Companion — Feather source

Personal AltStore/Feather source for the **Unleashed** iOS app (Flipper Zero companion).

## Add to Feather
Feather → Sources → add:
```
https://raw.githubusercontent.com/squazaryu/unleashed-companion/main/apps.json
```
Feather signs it with your certificate on install and updates it when a new version is published.
